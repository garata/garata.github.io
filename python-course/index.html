Monty Python heritage

A part from referring to to the very famous british comedy group, when people say they Python in relation to programming they either mean the Python programming language or they mean the tool that can translate some text (code) written in the Python programming language to the language a computer can actually understand. On Microsoft Windows this is the python.exe you need to install. On Linux/Mac it is usally called python or python3. CPython is the tool that translates a programming language for the computer is eiter called a compiler or an interpreter. CPython is both the compiler and the interpreter.

Newby minimal requirements

The bare minimum requirements for this course are two pieces of software: A text editor in which you can write the program and a compiler or interpreter that can translate this program to the computer.

Basically you can use any text editor to write Python code, but, in order to obtain a good syntax highlighting, the IDE program should provide intellisense as well and, in most cases, they should be able to understand what kind of objects are instantiated in source code showing the available methods and their parameters. Even better, they provide powerful debuggers.

  • A complete list of code editor where we can write programs in Python language can be found here IntegratedDevelopmentEnvironments - Python WiKi
  • A Python 3 compiler or interpreter that can translate sources text to the language of the computer is the best way to go. In fact, Python has two major lines the version 2.x and the version 3.x, but withou further ado the latter should always be used if possible. A good piece of skill for a daily Python programmer is the ability to be able to recognize that the code was originally written for Python 2 and you need to be able to make the adjustments.

Installing Python 3 on Linux

On Linux you usually have Python 2 installed in /usr/bin/python and Python 3 installed in /usr/bin/python3. Depending on your distribution you can use yum or apt-get package manager to update your Linux distribution. 

which python3 # check where python3 executable is located
sudo apt-get install python3 # python3 linux debian style installation
sudo yum install python3 # python3 linux redhat style installation

Installing Python 3 on Apple Mac OSX

On Mac OSX to check if Python 2 is installed in /usr/bin/python and Python 3 is installed in /usr/bin/python3

which python2 # check where python2 executable is located
which python3 # check where python3 executable is located

Installing Python 3 on Microsoft Windows

Microsoft Windows system generally does not offer pre-installed Python, but this situation may be easily fixed using an up to date MSI installer took from Using Python on Windows official Python documentation page. Nowaday, Microsoft offers another interesting approach named as Windows Subsystem for Linux.

Python main function

The Python interpreter does not make assumptions about predefined function name entry point, it executes scripts starting at the top of the file and for such reason Python programmers adopt convention naming and special variables. Thus in Python scripts, it happens to see main function definition and a conditional statement. Assuming that the script is executed on command line you can use the if __name__ == "__main__" idiom to determine the execution context and conditionally run main() only when __name__ is equal to "__main__". Simply put the main() function should reimain slim containing calls to others functions and minimum to no special logic you can delegate to other functions and modules. Python uses indentation for blocks instead of curly braces, and it uses the colon : to start a block.

def main(): print("Hello World!") if __name__ == "__main__": main()

Python pieces of code

Python interpreter differentiates among three type of pieces of code. Thus, __name__ idiom takes on different values depending on how you executed your Python file.

  • File: Typically, a Python file is any file that contains code. Most Python files have the extension .py.
  • Script: A Python script is a file that you intend to execute from the command line to accomplish a task. "__main__" when the file is executed from the command line or with python -m (to execute a package's __main__.py file)
  • Module: A Python module is a file that you intend to import from within another module or a script, or from the interactive interpreter. "__main__" when the module is imported contains name of the module

Handling user input

When the Python interpreter exeute a command line script you can interact with standard input to acquire values from the user, and you run into an exception error you can leave this occurence unhandled or wrap it into try ... except: ... [else:] block. As shown into the example, the try and except block in this Python snippet is used to catch and handle exceptions. Python interpreter executes code following the try statement as a monitored for run time errors part of the program. The code that follows the except statement is how the program should for example response to a Python Standard Library in the preceding try clause.

def main(): try: a = int(input("Enter the first input : ") or "1") b = int(input("Enter the second input : ") or "2") print(int(a) + int(b)) except EOFError: print("This console does not support input") main()

For loop

For loops are useful for iterating over a sequence of data. To iterate means to step through each element one at a time. For loops are also useful for repeating a process. The range(start, stop[, step]) function works similarly to the randrange function in that the first number is inclusive and the second number is exclusive.

  • The start argument is a starting number of the sequence. By default, this lower limit, it starts with 0 if not specified.
  • The stop argument is an upper limit. The range() function considers stop as less than criteria and doesn't include this number in the result.
  • The step is a difference between each number in the result. The default value of the step is 1 if not specified.

range() does not generate all numbers at once. In every iteration of for loop, range() generates the next number and assigns it to the iterator variable of loop

# range is from 0 to 4 for i in range(5): print(i, end=', ') print('\n') # range is from 1 to 6 start = 1 stop = 5 step = 1 stop +=step #now stop is 6 for i in range(start, stop, step): print(i, end=', ')

For-in loop are also useful to iterate over a well defined list of element values. In python terminology iterable is the term used referring to data structure you ca go through elements looping.

text = 'hallo world' for c in text: print(c) for c in ["aaa", "bbb", "ccc", "ddd", "eee", "fff"]: print(c)

While loop

Python while loop is a good alternative to for loop when you want to evaluate a complex or medium-complex condition constraining loop break. The break instruction is available in Python allowing to exit loops when break out condion applies. Thus, since in Python language do ... while is not avaliable, you have to implement the same logic with infinite loop and one or more break out conditions. But there's more, Break out from multi-level loops is not feasible in Python and you have to implement a similar solution refactoring your algorithms with functions call.

import random count = 1 randNum = random.randint(0, 10) # range random count value while randNum is not 5: # as soon as randNum == 5 the loop will terminate count += 1 # increment counter by one # in order to avoid an infinite loop randomly reinit sentinel value randNum = random.randint(0, 10) # we’re outside of the for loop print("It took %s tries to get 5 as integer random value." % str(count))

List copy

  • Presumably there is one list in the memory and two pointers to it.
  • When you want to make a dangling copy the way is to use the slice syntax.
  • When we copy just the address Python is making a shallow copy; anyway copy.deepcopy is the way to go to deep copy a list.
lst1 = ['aaa', 'bbb', 'ccc', 'ddd'] lst2 = lst1 lst1[0] = 'zzz' print(lst1) # ['zzz', 'bbb', 'ccc', 'ddd'] print(lst2) # ['zzz', 'bbb', 'ccc', 'ddd'] lst3 = lst2[:] lst2[0] = 'xxx' print(lst2) # ['zzz', 'bbb', 'ccc', 'ddd'] print(lst3) # ['xxx', 'bbb', 'ccc', 'ddd']
from copy import deepcopy lst1 = ['aaa', 'bbb', 'ccc', 'ddd'] lst2 = deepcopy(lst2) lst1[0] = 'zzz' print(lst1) # ['zzz', 'bbb', 'ccc', 'ddd'] print(lst2) # ['aaa', 'bbb', 'ccc', 'ddd']

Collections dequeue

Deque (Doubly Ended Queue) in Python is implemented using the module "collections". Deque is preferred over list in the cases where we need quicker append and pop operations from both the ends of container, as deque provides an O(1) time complexity for append and pop operations as compared to list which provides O(n) time complexity. When required Deque could be of fixed size passing the second argument like in this example: = deque([], maxlen = ?)

from collections import deque lst = deque(['aaa', 'bbb']) print(type(lst)) # print(lst) # deque(['bbb', 'aaa']) lst.append('zzz') print(lst) # deque(['aaa', 'bbb', 'zzz']) print(len(lst)) # 3 lst.append('zzz') print(lst) # deque(['aaa', 'bbb', 'zzz', 'zzz']) nxt = lst.popleft() print(nxt) # 'aaa' print(lst) # deque(['bbb', 'zzz', 'zzz']) print(len(lst)) # 3 if lst: # check in boolean context is list is empty print("The queue has elements") else: print("The queue is empty") # define a fixed list Deque instance lst2 = deque([], maxlen = 3) print(len(lst2)) # 0 print(lst2.maxlen) # 3 lst2.append("aaa") lst2.append("bbb") lst2.append("ccc") print(lst2) # deque(['aaa', 'bbb', 'ccc'], maxlen=3) lst2.append("zzz") # Automatically removes the left-most (first) element print(lst2) # deque(['aaa', 'bbb', 'zzz'], maxlen=3)

List as stack

from collections import deque lst = deque() lst.append("aaa") lst.append("bbb") lst.append("ccc") while stack: val = stack.pop() print(val) # ccc # bbb # aaa

Sort w/ sorted

  • The sort() method will sort a list in-place and return None.
  • The built-in sorted() function will return the sorted list and leave the original list intact.
lst = ['aaa', 'ddd', 'ccc', 'bbb'] print(lst) # ['aaa', 'ddd', 'ccc', 'bbb'] s = sorted(lst) print(s) # ['aaa', 'bbb', 'ccc', 'ddd'] print(lst) # ['aaa', 'ddd', 'ccc', 'bbb'] r = sorted(lst, reverse=True, key=len) print(r) # ['ddd', 'ccc', 'bbb', 'aaa'] print(lst) # ['aaa', 'ddd', 'ccc', 'bbb']
