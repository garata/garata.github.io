Monty Python heritage

A part from referring to to the very famous british comedy group, when people say they Python in relation to programming they either mean the Python programming language or they mean the tool that can translate some text (code) written in the Python programming language to the language a computer can actually understand. On Microsoft Windows this is the python.exe you need to install. On Linux/Mac it is usally called python or python3. CPython is the tool that translates a programming language for the computer is eiter called a compiler or an interpreter.

Newby minimal requirements

The bare minimum requirements for this course are two pieces of software: A text editor in which you can write the program and a compiler or interpreter that can translate this program to the computer.

Basically you can use any text editor to write Python code, but, in order to obtain a good syntax highlighting, the IDE program should provide intellisense as well and, in most cases, they should be able to understand what kind of objects are instantiated in source code showing the available methods and their parameters. Even better, they provide powerful debuggers.

  • A complete list of code editor where we can write programs in Python language can be found here IntegratedDevelopmentEnvironments - Python WiKi
  • A Python 3 compiler or interpreter that can translate sources text to the language of the computer is the best way to go. In fact, Python has two major lines the version 2.x and the version 3.x, but withou further ado the latter should always be used if possible. A good piece of skill for a daily Python programmer is the ability to be able to recognize that the code was originally written for Python 2 and you need to be able to make the adjustments.

Installing Python 3 on Linux

On Linux you usually have Python 2 installed in /usr/bin/python and Python 3 installed in /usr/bin/python3. Depending on your distribution you can use yum or apt-get package manager to update your Linux distribution.

which python3 # check where python3 executable is located
sudo apt-get install python3 # python3 linux debian style installation
sudo yum install python3 # python3 linux redhat style installation

Installing Python 3 on Apple Mac OSX

On Mac OSX to check if Python 2 is installed in /usr/bin/python and Python 3 is installed in /usr/bin/python3

which python2 # check where python2 executable is located
which python3 # check where python3 executable is located

Installing Python 3 on Microsoft Windows

Microsoft Windows system generally does not offer pre-installed Python, but this situation may be easily fixed using an up to date MSI installer took from Using Python on Windows official Python documentation page. How to insert comments

Python programmer does use two kinds of comment relying on the fact that comments are not run by the Python interpreter, and are used to remarks important concept for humans. As said, there are two types of comments: block and single line. A block comment is stated with triple quotes, either single or double, and closed with the same quotes.

Python program Literals and Value Types

Python basically offers type checking as runtime feature.

print( type(123) ) # int print( type(3.14) ) # float print( type("Hello World") ) # str print( type(None) ) # NoneType print( type(True) ) # bool, accepts False as well print( type([]) ) # list print( type({}) ) # dict print( type(3.14) ) # floating point (2**53 binary floating-point)

Python program Variables

Python has five standard Data Types: Numbers, Strings, Lists, Tuples, and Dictionaries. The Python type system dynamically initilize the Data Type on its initialization.

Within the Numbers data type there are subtypes: int, long, float, and complex.

  • int -> whole numbers (integers)
  • long -> really big integers
  • float -> decimal numbers
  • complex -> complex numbers
